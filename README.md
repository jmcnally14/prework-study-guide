# <Jordan's Prework Study Guide>

## Description

This Prework Study Guide was created for boot camp students who were going through the assigned Prework prior to class starting. It contains relevant notes pertaining to HTML, CSS, Git, and Javascript that any student may want to know about during their course of study. At the completion of the Prework this study guide should be able to set the student up for success to assist them through their time in the coding bootcamp.

N/A

## Usage

To use this Prework Study Guide, you can review the notes in each section. Each section is broken out to cover each individual topic that was reviewed during the Prework process for the class including HTML, CSS, Git, and Javascript. For suggestions on what to study you can utilize the ChromeDevTools console to generate each time what you should study at random. If you are unfamilar on how to access the ChromeDevTools on a Windows Computer using your keyboard hit Ctrl + Shift + I which will bring up the ChromeDevTools Screen. Navigate to the Console tab located on the lefthand side of your screen to view the Console, unless it auto populates to it. Each time you refresh the browser a new section on what to study first with generate for you.
